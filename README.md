# Tugas CRUD - Data Gebetan

Aplikasi berbasis web sederhana untuk mengelola *Data Gebetan (Crush List)* yang dibangun menggunakan Framework Laravel dan PHP. Repository ini dibuat untuk memenuhi tugas mata kuliah pemrograman web.

## Fitur Utama
Aplikasi ini memiliki fungsi CRUD dasar:
- **Create (Tambah)**: Menambahkan data gebetan baru (Nama, Alasan Suka, Status Hubungan, Level Ketertarikan).
- **Read (Tampil)**: Menampilkan daftar data gebetan dalam bentuk tabel HTML klasik.
- **Update (Edit)**: Mengubah detail data gebetan yang sudah ada di database.
- **Delete (Hapus)**: Menghapus data gebetan dari daftar tabel.

## Teknologi yang Digunakan
- **Backend**: PHP 8.x, Framework Laravel
- **Database**: MySQL
- **Frontend**: HTML Dasar (tanpa styling CSS yang rumit)
